## Step 1: Account Setup & Organization Creation

### Initial Login
1. **Sign up** at your TalentPatriot instance
2. **Choose your authentication method**: Google, Microsoft, or email
3. **Complete the 5-step onboarding wizard**:
   - Organization details
   - Team size and industry
   - Primary hiring needs
   - Integration preferences
   - Welcome tour

### Organization Configuration
```
Organization Name: [Your Company]
Industry: [Select from dropdown]
Team Size: [5-500 employees]

```

## Step 2: Post Your First Job

### Job Creation Wizard
1. **Navigate to Jobs** ‚Üí "Post New Job"
2. **Fill in basic details**:
   - Job title
   - Department/Client
   - Employment type (Full-time, Part-time, Contract)
   - Location (Remote, Hybrid, On-site)

3. **Job Description**:
   - Role overview
   - Key responsibilities
   - Required qualifications
   - Preferred skills
   - Compensation range (optional)

4. **Application Settings**:
   - Enable AI resume parsing
   - Configure required vs optional fields
   - Set application deadline

### Example Job Posting
```

Client: Internal - Engineering Team
Type: Full-time
Location: Remote (US/Canada)
Status: Draft ‚Üí Open

Description:
We're looking for a Senior Frontend Developer to join our growing engineering team...

- 5+ years React experience
- TypeScript proficiency
- Modern CSS/Tailwind experience
```

## Step 3: Configure Your Careers Page

### Branding Setup
1. **Go to Settings** ‚Üí "Careers Page"
2. **Upload company logo** (recommended: 200x80px)
3. **Set brand colors** and company description
4. **Configure contact information**

### Public URL
Your careers page will be available at:
- `yourcompany.talentpatriot.com/careers` (subdomain)
- Or `app.talentpatriot.com/org/yourcompany/careers` (path-based)

### Custom Domain (Optional)
For enterprise customers, configure custom domains like:
`careers.yourcompany.com`

## Step 5: Understanding the Pipeline

### Default Pipeline Stages
When you create a job, TalentPatriot automatically creates these pipeline stages:

1. **Applied** - New applications
2. **Phone Screen** - Initial screening calls
3. **Interview** - Formal interviews
4. **Technical** - Technical assessments
5. **Offer** - Offer extended
6. **Hired** - Successfully hired
7. **Rejected** - Not moving forward

### Customizing Pipeline Stages
- **Add custom stages** that match your process
- **Reorder stages** by dragging and dropping
- **Set stage colors** for visual organization
- **Configure automatic notifications** for stage changes
